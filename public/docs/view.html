View

Delete any code in the code editor, run your blank program, hit the fit button, and zoom out a bit. Do you see that faint rectangular outline? That rectangular outline marks the view.

Anything inside the view will be visible when you bring your design into other programs. Anything outside the view will be cut off.

For example, consider this Twoville program:

circle color = :yellow center = [0, 50] radius = 50

Half the circle lies outside the view. When you export the SVG file and embed it in a web page, it looks like this:

Within Twoville itself, you can see both inside and outside the view.

By default, the view's bottom-left corner is (0, 0) and its top-right corner is (100, 100). If you want a shape to be visible, it must reside in this coordinate system—or you must change the coordinate system.

To change the coordinate system, you set properties of the builtin view object. It has properties corner, center, and size, just like rectangles.

In the following code we set the size of the view to fit around the magenta rectangle.

view.size = [40, 30] rectangle corner = [5, 5] size = [30, 20] color = :magenta

If you want to set multiple properties, the view object can be used with a block:

view size = [40, 30] center = [20, 15] rectangle corner = [5, 5] size = [30, 20] color = :magenta
