Bézier Curves

When you add a line node to a path, you get a straight line segment. When you add an arc node, you get a circular arc. You can make a lot of interesting shapes by mixing these together. But what about something like this?

There aren't any straight lines on that shape, and the curves don't look very circular. In fact, this path was drawn using neither of these node types. This shape is made of quadratic Bézier curves. (Bézier is a French name that is pronounced like BEZ yay.)

To plot a quadratic Bézier curve, you need its two endpoints and a third point that acts like a gravitational force pulling the line connecting the two endpoints toward it. This gravitational point is called a control point.

Consider this code that plots an arch:

with path() stroke.color = :pink stroke.size = 2 opacity = 0 jump().position = [20, 0] with quadratic() position = [80, 0] control = [50, 110]

We issue a jump node to establish the starting endpoint at the bottom left of the viewport, and then we issue a quadratic curve that ends at the bottom right of the viewport. Instead of a straight line connecting them, the control point [50, 100] drags the curve upward.
