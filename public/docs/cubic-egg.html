Cubic Egg

What if I said I had an uncle who had a chicken that laid cubic eggs? Hard to believe, probably. Well, you are about to lay a cubic egg. Hard to believe, probably.

The word cubic in this case doesn't mean cube-shaped. Recall that quadratic Bézier curves have just one control point influencing the curvature. Cubic Bézier curves have two control points. The first black hole controls the curvature at the start of the curve. The second black hole controls the curvature at the end of the curve.

Let's model an egg using two cubic curves. Start by placing the path's first node at the bottom center of the viewport:

with path() color = [1, 0.9, 0.9] stroke.color = [0.9, 0.8, 0.8] stroke.size = 2 jump().position = [50, 1]

You won't be able to see anything when you run this. Next add a cubic node that ends at the top center and has its two control points tugging the curve to the left:

with path() color = [1, 0.9, 0.9] stroke.color = [0.9, 0.8, 0.8] stroke.size = 2 jump().position = [50, 1] with cubic() position = [50, 99] control1 = [-10, 1] control2 = [20, 99]

Run this and you'll see a semi-egg. Place the cursor inside the cubic block to show the handles for the three properties. Drag the control point handles around to see how they change the curvature.

To complete the egg, add a second cubic node that ends where we started. Just as we saw with quadratic curves, we can leave off a control point to make a smooth transition between the two segments. For cubic curves, we can only omit control1. By dropping control1 and making control2 mirror its partner on the other side, we get a nice symmetric egg:

with path() color = [1, 0.9, 0.9] stroke.color = [0.9, 0.8, 0.8] stroke.size = 2 jump().position = [50, 1] with cubic() position = [50, 99] control1 = [-10, 1] control2 = [20, 99] with cubic() position = [50, 1] control2 = [110, 1]

What a lovely egg. My uncle's chicken would be proud.
