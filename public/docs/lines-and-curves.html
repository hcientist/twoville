Lines and Curves

What do you call a shape that's half square and half circle? You don't call it anything, because it doesn't have a name. We often think of shapes as pure: all straight lines like a rectangle or all curves like a circle. But some shapes are a mix.

You can create mixed shapes using the path command. A path is built up out of nodes. Some nodes plot straight line segments, and others are curved. The many possible node types of path make it the most complex command in all of Twoville. We'll visit just three of them in this lesson.

To start tracing out a path, you issue a go node and set its position property, like this:

path go.position = :zero

If you run this code, you'll see errors. Let's add some color and stroke:

path color = :soggy stroke color = :black weight = 2 go.position = :zero

Run this code, and you'll see no errors and no nothing. You need at least two nodes in a path before we see any stroke. You need at least three before we see any of its fill color. To create a straight edge, issue a line node:

path color = :soggy stroke color = :black weight = 2 go.position = :zero line.position = [100, 0]

Do you see a line? I hope so. Let's make another line segment:

path color = :soggy stroke color = :black weight = 2 go.position = :zero line.position = [100, 0] line.position = [100, 50]

Do you see two edges and a filled triangle? If so, let's proceed by adding a semi-circular curve across the top. The arc draws a curve defined by several properties. This code draws an arc starting from [100, 50] and going 180 degrees around a circle centered at [50, 50]:

path color = :soggy stroke color = :black weight = 2 go.position = :zero line.position = [100, 0] line.position = [100, 50] arc center = [50, 50] degrees = 180

The arc has its center pinned down. There may come a day where you find it easier to think about where the curve ends rather than its center. You can use the position property on that day, like this:

path color = :soggy stroke color = :black weight = 2 go.position = :zero line.position = [100, 0] line.position = [100, 50] arc position = [0, 50] degrees = 180

The result is the same. Just a heads-up. You can't set both position and center.

How does the shape look now? There's a piece of stroke missing on the left side, you say? Let's fix that. Add a back node to the path automatically insert a line segment connect the last node to the first:

path color = :soggy stroke color = :black weight = 1 go.position = :zero line.position = [100, 0] line.position = [100, 50] arc center = [50, 50] degrees = 180 back

There's our half-square, half-circle. You could call it a tombstone, maybe.
