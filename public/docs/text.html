Text

Sooner or later, you will need to make a treasure map. Treasure maps are full of pictures of trees and piranhas and rocks, but they also have text labels. To draw text in Twoville, use the text command. There are several properties that you need to set on a text object:

  • the message containing the letters, numbers, and other symbols of text
  • the color of the text
  • the size of the text
  • the position of the text

In this program we mark the location of some gold using the text command and a black dot:

with viewport size = [100, 40] corner = [0, 30] position = [50, 50] with circle() center = position radius = 3 color = :black with text() message = "gold" size = 16 position = position color = :orange

Run it. Notice how the text and circle have the same position. The text is anchored to this position just as your pirate ship is anchored to its place in the sea. By default, the point of the text that serves as the anchor is the bottom center of the letters. You'll occasionally want the anchor to be in a different spot on the text. You can set the location with the anchor property. The anchor is a list of two alignment constants. The first element is the horizontal position and can be :west, :center, or :east. The second element is the vertical position and can be :north, :center, or :south.

All told, there are these nine possible anchorings:

  • anchor = [:west, :north]
  • anchor = [:center, :north]
  • anchor = [:east, :north]
  • anchor = [:west, :center]
  • anchor = [:center, :center]
  • anchor = [:east, :center]
  • anchor = [:west, :south]
  • anchor = [:center, :south]
  • anchor = [:east, :south]

The text command also works on drawings that aren't treasure maps.
