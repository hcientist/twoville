Keyboard Shortcuts

Tired of clicking? Try using your keyboard to trigger some of Twoville's most frequently used operations.

Apple Keyboards

Run ⌘Enter
Fit ⌘.
Open ⌘o
Save ⌘s
Select all ⌘a
Bigger font ⌘=
Smaller font ⌘-

Non-Apple Keyboards

Run CTRLEnter
Fit CTRL.
Open CTRLo
Save CTRLs
Select all CTRLa
Bigger font CTRL=
Smaller font CTRL-
