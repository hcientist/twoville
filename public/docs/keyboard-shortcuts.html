Tired of clicking? Try using your keyboard to trigger some of Twoville's most frequently used operations.

Apple Keyboards

Operation Shortcut
Run ⌘Enter
Fit ⌘.
Open ⌘o
Save ⌘s
Select all ⌘a
Increase font size ⌘=
Decrease font size ⌘-

Non-Apple Keyboards

Operation Shortcut
Run CTRLEnter
Fit CTRL.
Open CTRLo
Save CTRLs
Select all CTRLa
Increase font size CTRL=
Decrease font size CTRL-
