Translate

Suppose you've got a shape but discover it's in the wrong spot. You have two choices to get it to the right spot. Either you go back and update every position of the shape, which may be a daunting task for some shapes, or you add a translate modifier to your shape. The translate modifier has an offset property that will be added automatically to all positions.

Consider this square, whose center is shifted right and down by translate:

rectangle center = [50, 50] size = [30, 30] color = :tomato translate offset = [10, -20]

The translate modifier can be used to animate a shape that slides around:

rectangle center = [50, 50] size = [30, 30] color = :tomato translate 0 -> t t -> 100 offset = [10, -20] t -> 50 -> t offset = [-10, 20]
