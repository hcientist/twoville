Constants

You've seen that there are many colors in Twoville that have been named. These names can make the code easier to read. When you set a color property to [1, 0.388, 0.278], most of us would have no idea what color would appear. But when you use :tomato, we can make a reasonable prediction.

When you use a name of the form :name, you are using a constant that is built in to the Twoville language. There are a number of constants. Some are used only with the certain commands, and you will learn about them when you learn about the commands. Others are more general, like these:

  • :zero2, which is [0, 0]
  • :up, which is [0, 1]
  • :down, which is [0, -1]
  • :right, which is [1, 0]
  • :left, which is [-1, 0]

In the code you write, you are free to choose between the named form and the value form. In these notes, you will find us favoring the names. For example, we will often use :zero2 for positions.
