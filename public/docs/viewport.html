Viewport

Delete any code in the code editor, run your blank program, hit the fit button, and zoom out a bit. Do you see that faint rectangular outline? That rectangular outline marks the viewport.

Anything inside the viewport will be visible when you bring your design into other programs. Anything outside the viewport will be cut off.

For example, consider this Twoville program:

with circle() color = :yellow center = [0, 50] radius = 50

Half the circle lies outside the viewport. When you export the SVG file and embed it in a web page, it looks like this:

Within Twoville itself, you can see both inside and outside the viewport.

By default, the viewport's bottom-left corner is (0, 0) and its top-right corner is (100, 100). If you want a shape to be visible, it must reside in this coordinate system—or you must change the coordinate system.

To change the coordinate system, you set properties of the builtin viewport object. It has properties corner, center, and size, just like rectangles.

In the following code we set the size of the viewport to fit around the magenta rectangle.

viewport.size = [40, 30] with rectangle() corner = [5, 5] size = [30, 20] color = :magenta

If you want to set multiple properties, the viewport object can be used in a with block.

with viewport size = [40, 30] center = [20, 15] with rectangle() corner = [5, 5] size = [30, 20] color = :magenta

Note that you do not put any parentheses after viewport like you did with rectangle and circle. The parentheses are used when you are making something new. You are not making a new viewport; you are tweaking the existing one.

Grid Lines

You may find it useful to plot a grid on your viewport, which you can do by setting the grid property to a list of two numbers. The list [1, 1] plots a grid line on every whole number x-coordinate and y-coordinate. The list [2, 5] plots a line on every even x-coordinate and every y-coordinate that's a multiple of 5.

This viewport plots a grid line every 2 units on both axes:

with viewport size = [10, 10] grid = [2, 2]
