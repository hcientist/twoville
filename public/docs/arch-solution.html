Arch Solution

Here is one way to create the arch design:

path stroke color = :pink weight = 2 color = [1, 0.75, 1] go.position = [20, 0] quadratic position = [80, 0] control = [50, 110] line.position = [100, 0] quadratic position = :zero control = [50, 150] back

After the original parabola, a line extends to the bottom-right corner. Then a quadratic curve jumps all the way to the bottom-left corner. A back node connects the last node to the first. Finally, the opacity is removed and the whole thing is filled with a lighter pink.
