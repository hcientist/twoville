Line

You've made circles. You've made rectangles. These shapes are predictable, having just a few properties. From here on out, however, you'll make much wilder shapes by connecting vertices. The first of these that we'll discuss is line.

To draw a line, you call its vertex function twice and set each vertex's position property. You also set its size property to control its thickness and its color property. For example, this code plots a diagonal line across the viewport.

line size = 5 color = :blue vertex.position = :zero vertex.position = [100, 100]
