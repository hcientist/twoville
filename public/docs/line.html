Line

You've made circles. You've made rectangles. These shapes are predictable, having just a few properties. From here on out, however, we'll make much wilder shapes by connecting corners or vertices. The first of these that we'll discuss is line.

To draw a line, you call its vertex function twice and set each vertex's position property. You also set its size property to control its thickness and its color property. For example, this code plots a diagonal line across the viewport.

with line() size = 5 color = :blue vertex().position = :zero2 vertex().position = [100, 100]
