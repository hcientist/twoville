Fill and Stroke

You've made a circle that was colored cornflower and a rectangle that was colored crimson. Did you try using other colors?

Many colors are named: :red, :yellow, :magenta, :blue, and so on. Not all of them are named, however. You can get at these unnamed colors using a color list.

Color lists are mixtures of three primary colors: red, green, and blue. They have the form [red, green, blue], where red, green, and blue are numbers between 0 and 1.

The symbol :red is shorthand for [1, 0, 0]. The symbol :green is shorthand for [0, 1, 0]. The symbol :blue is shorthand for [0, 0, 1].

To mix together a lot of red, some green, and no blue, we'd write [1, 0.5, 0]. What color is the world that uses this list?

with circle() radius = 40 center = [50, 50] color = [1, 0.5, 0]

Color lists are definitely something that you should experiment with.

A shape's color property sets the shape's fill color, which is the color that appears within the shape's perimeter. To set the color of the shape's perimeter itself, you set the stroke color. You also need to set the stroke size, which is 0 by default. Tweak your program to look like this.

with circle() radius = 40 center = [50, 50] color = [1, 0.5, 0] stroke.color = :black stroke.size = 1
