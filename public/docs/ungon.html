Ungon

Corners are alright. But what if you don't like them? Well, you might try an ungon. Ungons are like polygons, but their vertices are rounded off. This means they are "not polygons."

Here's the pentagonal house again, but this time it uses the ungon command:

with ungon() rounding = 0 color = :orange vertex().position = [10, 10] vertex().position = [90, 10] vertex().position = [90, 60] vertex().position = [50, 90] vertex().position = [10, 60]

Ungons have one additional over polygons: rounding. When rounding is 0, the corners are only slightly rounded. When rounding is 1, the corners are really rounded. If you use a number outside of [0, 1], be prepared for strange effects.
