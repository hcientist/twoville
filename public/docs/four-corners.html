Four Corners

Sometimes people talk of the "four corners" of the world. Worlds generally don't have corners. But you can make one that satisfies these confused people.

Use the rectangle command to make a four-corner world. Type this code into the editor and run it.

with rectangle()
  size = [20, 10]
  corner = [0, 0]
  color = :crimson

The rectangle command is similar to circle, but it has some different properties. Both have color, but instead of radius, you set its size. Instead of center, you set its corner.

Wait. That last sentence stretches the truth. Try switching corner to center. 

with rectangle()
  size = [20, 10]
  center = [0, 0]
  color = :crimson

Zoom out and pan to see the whole rectangle.

The whole truth is that you can use either center or corner for rectangles—but you can't use both.

Here's a challenge for you. Try changing the coordinates of center so that the rectangle sits right in the middle of the viewport.

What coordinates did you use? What coordinates would you use to center the rectangle if you were using corner?
